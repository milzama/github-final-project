# Kalkulator Bunga Sederhana

Proyek ini adalah aplikasi berbasis skrip Bash sederhana untuk menghitung bunga tahunan berdasarkan nilai pokok, tingkat bunga tahunan, dan periode waktu dalam tahun.

## Deskripsi
Skrip ini dibuat sebagai bagian dari proyek akhir GitHub untuk mendemonstrasikan pemahaman tentang kontrol versi, dokumentasi, dan skrip shell dasar.

## Cara Penggunaan
1. Berikan izin eksekusi pada skrip: `chmod +x simple-interest.sh`
2. Jalankan skrip: `./simple-interest.sh`
3. Masukkan input saat diminta.
